# vgg19_maya_als_classifier
Implementing terrain visualization and spatial analysis using the Kokalj et al. (2023) dataset. This project explores the integration of advanced LiDAR processing techniques with modern geoinformation workflows to enhance feature recognition in digital terrain models.

## About the Project
This project is conducted as part of my **Bachelor of Science Thesis** at **Istanbul Technical University (ITU)**, Department of **Geomatics Engineering**.

### Objective
The primary aim is to process and visualize spatial data to identify micro-topographic features. By utilizing the datasets and methodologies proposed by **Kokalj et al. (2023)**, this research compares different visualization techniques (such as Sky-View Factor, Openness, and local relief models) to determine their impact on topographic interpretation.
